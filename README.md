## Getting Started

### Requirements

- Dart: 3.4.0
- Flutter SDK: 3.21.0

### Install

- WARN: If you already installed `make`, you could omit this step.

- Install make:
    - Run `choco install make`
  
 ### Config and run app

- cd to root folder of project
- Run `make gen_env`
- Run `make pub_get`
- Run `make run_dev`
- Run & Enjoy!
